# 京都市の定期報告（条例による追加項目）

京都市の届出住宅は、国の定期報告4項目に **市条例で項目が追加** されており、提出方法も独自様式が用意されている。「国の4項目だけ出せばよい」は京都市では **誤り**。

根拠: 京都市住宅宿泊事業の適正な運営を確保するための措置に関する条例 第16条第1項＋住宅宿泊事業法第14条。

主な出典: 京都市「住宅宿泊事業 届出の手引」（令和8年4月1日版、定期報告は「定期報告について」の章 P17-1〜）: https://www.city.kyoto.lg.jp/hokenfukushi/cmsfiles/contents/0000287/287839/tebikizentai.pdf
京都市民泊ポータル: https://minpakuportal.city.kyoto.lg.jp/ / 京都市の独自ルール解説: https://www.city.kyoto.lg.jp/hokenfukushi/page/0000233644.html
最終確認日 2026-07-02。**手引・様式・連絡先は改定されることがある** — 提出前に最新版の手引を必ず確認すること。

---

## 1. 京都市かどうかの判定

次のいずれかに該当したら京都市ルールを適用する（設定ファイル `minpaku-config.yaml` の該当物件で `kyoto: true`）:

- 届出番号の数字の先頭2桁が **26**（京都府。京都市の公表一覧は全件 M26 で始まる）で、所在地が京都市内
- 物件の所在自治体（municipality）が京都市

京都府内でも京都市外の物件には京都市条例は適用されない（その自治体の窓口で独自ルールを確認: https://www.mlit.go.jp/kankocho/minpaku/municipality.html ）。

## 2. 報告する内容（手引の原文どおり）

毎年2月・4月・6月・8月・10月・12月の15日までに、それぞれの月の前2月分を京都市長に報告する（国と同じスケジュール）。宿泊させた日数が0日でも必ず報告が必要。

**⑴ 届出住宅における宿泊者の状況（毎回）** — 国の4項目:
- ア 届出住宅に人を宿泊させた日数
- イ 宿泊者数（「届出住宅に宿泊した実際の人数を該当期間で足し合わせた数。同一人物が同じ届出住宅に連続して宿泊した場合は、１人と数える。」）
- ウ 延べ宿泊者数（「各日の全宿泊者数を該当期間で足し合わせた数」）
- エ 国籍別の宿泊者数の内訳

**⑵ 届出住宅の周辺住民からの苦情の状況（毎回）** — 京都市の追加項目:
- ア 苦情を受けた件数
- イ 苦情を受けた日時
- ウ 苦情の内容
- エ 苦情への対応の状況

苦情が0件の場合は「0件」と報告する（様式の別紙＝苦情内容等の明細は0件なら提出不要）。

**⑶ 廃棄物を適正に処理したことを証する書類（該当時のみ）**:
事業で生じた廃棄物を処理したときは、**処理した日以後で最初に行う定期報告と同時に**、適正処理を証する書類（収集運搬許可業者との契約書の写し、京都市クリーンセンターや処理業者の領収書の写し等）を提出する。

## 3. 苦情ログの記録（kujo-log.csv）

京都市物件では、報告期間中に受けた苦情を **期間フォルダ内の `kujo-log.csv`** に記録しておく。エンジン（`scripts/generate_report.py`）がこれを読み、苦情の件数・日時・内容・対応状況を出力に含める。

形式（1行 = 苦情1件、1行目はヘッダー固定）:

```csv
date,time,content,response
2026-06-14,23:10,宿泊客が深夜に騒いでいた,現地対応管理者から宿泊客に対して騒がないように注意を行った
2026-07-02,15:30,ごみ出しの曜日が守られていない,清掃担当者にごみ出しルールを再周知し翌日回収した
```

- `date`: 苦情を受けた日付（YYYY-MM-DD）。報告期間内の日付のみ記載する。
- `time`: 苦情を受けたおおよその時刻（HH:MM の24時間表記。不明なら空欄可）。
- `content`: 苦情の内容（日本語で簡潔に）。
- `response`: 苦情への対応状況（誰が・何をしたか）。
- 苦情が0件の期間は、ファイルを置かない（またはヘッダー行のみのファイル）でよい — エンジンは件数0として扱う。

京都市の様式の記入例では「現地対応管理者から宿泊客に対して騒がないように注意を行った」のように **対応の主体と行動** を書いている。同じ粒度で記録しておくと転記がそのまま済む。

なお京都市には現地対応管理者（駐在または概ね10分・道のり800m以内での駆けつけ）の設置義務があり、苦情対応はこの管理者が担うのが通常（京都市独自ルール。詳細は上記の独自ルールページ）。

## 4. 提出方法（京都市は独自様式）

京都市の定期報告は、市の **定期報告様式（第4号様式「届出住宅の宿泊日数、宿泊者数等及び苦情に係る報告書」、手引 17-6ページ参照）** に記入して提出する。様式は法第14条と条例第16条第1項の両方に基づく報告を1枚で兼ねる（様式冒頭:「住宅宿泊事業法第１４条及び京都市…条例第１６条第１項の規定により…報告をします」）。

様式の構成（エンジン出力からの転記対応）:
- 届出住宅の所在地・届出番号・報告対象の期間
- **a 宿泊日**: 2か月分のカレンダーに、宿泊させた日付へ○を付ける → エンジン出力の「宿泊日リスト」（クリックガイドと同じ日付一覧）をそのまま写す
- **b 宿泊者数・国籍別内訳**: 22区分（日本〜その他）の人数欄 → エンジン出力の22区分の数字をそのまま写す
- 宿泊日数（aの○の数）・宿泊者数（bの合計）・延べ人数
- 苦情: **件数は本紙に記入**し、内容欄は「別紙参照」として明細（日時・内容・対応状況）を**別紙に記載**する（手引の記入例どおり。0件なら別紙不要）

提出チャネル（いずれか）:
- **電子メール**: plb@city.kyoto.lg.jp — メールタイトルの先頭に **【定期報告】** と明記する
- **ファックス**: 075-251-7235
- **窓口持参・郵送**: 〒604-0835 京都市中京区御池通高倉西入高宮町200番地 千代田生命京都御池ビル2階 京都市保健福祉局医療衛生推進室 医療衛生センター 宿泊施設適正化担当（住宅宿泊事業審査担当）

画像データで提出する場合は記載事項が読める鮮明な画像で（不鮮明だと再提出になる）。

**メール・FAXで初めて提出する場合の事前登録**: 届出者本人からの提出であることを確認するため、事前登録届（手引 17-8ページ）＋本人確認書類（個人: 運転免許証・在留カード・マイナンバーカード表面等 / 法人: 定款の写し・印鑑登録証明書等）の事前提出が必要。ただし次のいずれかから提出する場合は事前登録不要:
1. 国の民泊制度運営システムに登録されたメールアドレス
2. 事業開始時に届け出たファックス番号
3. 事業者のホームページで公開している等、本人からの提出であることが明らかなメールアドレス・FAX番号

**未確認事項（fail-loud）**: 京都市の手引はこの独自様式での提出を案内しており、国の民泊制度運営システムへの入力で京都市の報告義務（苦情項目を含む）を満たせるのか、両方必要なのかは手引に明記がない。国のシステムには苦情欄が存在しないため、少なくとも苦情の状況は市の様式・チャネルでしか報告できない。**運用は医療衛生センター（上記連絡先）に確認すること。** それまでは市の様式での提出を既定とする。

## 5. 計算ルールの京都市の worked example（エンジンの検算に使える）

手引 17-3〜17-5 の例は国のルールと同じ算定を示しつつ、期間境界の扱いを明示している:

- 例1: 6月20日17時チェックイン → 6月24日10時チェックアウト = 宿泊日数 **4日**（国の留意事項と同一の例）。
- 例2: 3人が2泊3日＋その後5人が4泊5日 → 宿泊者数 **8人**。
- 例3: 同一人物Aが2泊3日で利用後、2日空けて再度3泊4日 → 宿泊者数 **2人**（連続でなければ回数分）。
- 例4: 3人×2泊＝6人、5人×4泊＝20人 → 延べ宿泊者数 **26人**。
- **例5（期間跨ぎ）**: 2人が5月30日〜6月2日に3泊4日 →「定期報告の末月と当初月に跨る場合、月末日～月初日の宿泊は末月分に含めてください」。つまり5月31日→6月1日の泊は5月分。
  - 6月15日締切分（4・5月期）: 宿泊日数2日・宿泊者数2人・延べ4人
  - 8月15日締切分（6・7月期）: 宿泊日数1日・宿泊者数2人・延べ2人
  - これは「泊はチェックイン日〜チェックアウト前日の各日付に帰属し、期間で切り詰める」というエンジンの算定と一致し、宿泊者数が跨ぎの両期間で1人ずつ数えられる国のFAQルールとも一致する。

**賃貸借契約の例外**（宿泊日数に算入しない長期滞在の京都市基準）: ①宿泊者との賃貸借契約書がある（契約日が初日）②契約日以降の連続滞在が暦月1か月以上 ③清掃等の衛生管理の主体が宿泊者側にある — の全てを満たす場合のみ賃貸借として扱える。仲介サイト経由で宿泊契約として始まった滞在は、長期というだけでは賃貸借にならない。詳細: https://www.city.kyoto.lg.jp/hokenfukushi/page/0000248047.html

## 6. 京都市のその他の注意（報告様式以外）

- **定期報告は届出者本人の義務**。手引（原文）:「定期報告は住宅宿泊事業者（届出者）の義務です。住宅宿泊管理業者が行うことはできません。」本スキルはあくまで転記用データの準備まで。
- **休業届は存在しない**。宿泊者がいなくても廃業までは報告義務が続く。長期間実施見込みがなければ廃業等届出書を提出する。
- **宿泊税は別の義務**。手引（原文）:「定期報告の提出期限は、宿泊税の納入申告期限とは必ずしも一致しませんので、御注意ください。」京都市は宿泊税の経営申告・納入義務が別途ある（本スキルの対象外）。
- 家主不在型は住居専用地域では原則1月15日正午〜3月16日正午のみ営業可（認定京町家事業を除く）、標識掲示・近隣事前周知など運営面の独自ルールも多い。報告様式とは別問題だが、京都市物件のオーナーは独自ルールページの確認を推奨。
